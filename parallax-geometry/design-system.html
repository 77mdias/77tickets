03. Motion Principles

Scroll Reveal

Elements enter from the bottom (y: 48px) with opacity fading from 0 to 1. The duration is 0.9s with "power3.out" easing.

Code Pattern

gsap.to(element, { 
  opacity: 1, 
  y: 0, 
  duration: 0.9, 
  ease: "power3.out" 
});

:root {
  /* Backgrounds */
  --background: #0a0a0a;          /* zinc-950 — page canvas */

  /* Text hierarchy */
  --foreground: #ffffff;           /* headings */
  --foreground-secondary: #d4d4d8; /* zinc-300 — body */
  --foreground-muted: #a1a1aa;     /* zinc-400 — captions */
  --foreground-subtle: #71717a;    /* zinc-500 — labels */

  /* Accent palette */
  --accent-blue: #60a5fa;          /* blue-400 — wireframe */
  --accent-amber: #fbbf24;         /* amber-400 — highlight */

  /* Surfaces */
  --surface-overlay: rgba(255,255,255,0.05);
  --surface-border: rgba(255,255,255,0.1);
  --surface-border-subtle: rgba(255,255,255,0.05);
}
